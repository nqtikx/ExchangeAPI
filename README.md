# The list of currently supported payment providers.

### Belarusian payment providers
- `ALFA` - BYN, RUB, USD, EUR - `OnRamp/OffRamp` **(card bind required)**, [instructions](./ALFA.md#alfa)
- `ASSIST` - BYN, RUB, USD, EUR - `OnRamp/OffRamp` **(card bind required)**, [instructions](./ASSIST.md#assist)
- `CA` - BYN, RUB, USD, EUR - `OnRamp/OffRamp` - bank transfer

### **Russian payment** providers

- `CARUSELL` - only RUB `OnRamp/OffRamp` Russian SBP mechanism, [instructions](./CARUSELL.md#carusell)
- `MTS` - only RUB - `OffRamp` **(card bind required)**, [instructions](./MTS.md#mts)
- `MTS` - only RUB - `OnRamp` - via bank mobile app, [instructions](./MTS.md#mts)
- `VTB` - only RUB - `OffRamp`, [instructions](./VTB.md#vtb)
- `SBER` - only RUB - `OnRamp`, - via bank mocile app, [instructions](./SBER.md#sber)
- `CA` - BYN, RUB, USD, EUR - `OnRamp/OffRamp` - bank transfer

### **CIS Countries** providers
- `ASSIST` - BYN, RUB, USD, EUR - `OnRamp/OffRamp VISA only` **(card bind required)**, [instructions](./ASSIST.md#assist)
- `CA` - BYN, RUB, USD, EUR - `OnRamp/OffRamp` - bank transfer
